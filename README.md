## Setup/Installation Requirements

1. Open Git Bash if on Windows and terminal if on Mac
2. Run the command

    ``Git clone https://github.com/leark/super-galactic-age-calculator.git``

3. Go inside pizza-parlor directory
4. Run the command

    ``npm init -y``

5. Run the command

    ``npm install``

6. Run the command

    ``npm run start``
